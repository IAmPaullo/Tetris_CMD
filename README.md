# Bem vindo!

Os comandos são simples:
**Enter**: Botão de acesso em todos os menus.
**Setas Esquerda**, **Baixo** e **Direita**: Controlam os Tetrominóes.
**Barra de Espaço**: Gira os Tetrominóes.
Divirta-se!!
